Você também pode retornar ao código HTML original. Para facilitar a edição posterior, há um botão de seleção que copia o código para a área de transferência. -->

      1. Copie seu código que deseja criptografar.
      2. Cole seu código na caixa de texto.
      3. Clique no botão "ENCRIPTAR".
      3. Clique no botão "PREVISUALIZAR".
      4. Selecione o texto criptografado.
      5. Cole o código criptografado no lugar do código original que você usou.

Recomenda-se que você use apenas esse método em locais que exigem maior segurança.
